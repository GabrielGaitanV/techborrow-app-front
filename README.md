# TechBorrow - Aplicación Web (Frontend)

Este repositorio contiene el código fuente del Frontend de la aplicación web **TechBorrow**, un sistema diseñado para la gestión, reserva y control de préstamos de equipos tecnológicos.

Este proyecto fue desarrollado como parte de la Actividad 4 de la asignatura de Construcción de Aplicaciones Web II.

## 🚀 Tecnologías y Herramientas Aplicadas
En este proyecto se evidencia la aplicación de los siguientes conceptos y tecnologías:
- **ReactJS:** Construcción de interfaces dinámicas y uso del Virtual DOM.
- **Vite:** Entorno de desarrollo rápido.
- **React Router DOM:** Manejo de rutas y navegación (SPA).
- **Axios:** Simulación de peticiones HTTP al servidor.
- **Hooks:** Uso de `useState`, `useEffect`, `useContext` y `useReducer` para el manejo de estados simples y complejos.
- **Context API:** Manejo del estado global de la sesión del usuario.
